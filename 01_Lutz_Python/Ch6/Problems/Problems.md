# Chaper 6 Problems
### 1.)  Consider the following; do they change the value printed for `A`?
  + A = "problem"
  + B = A

### 2.)  Consider the following; do they change the value printed for `A`?
+ A = ["problem"]
+ B = A
+ B = "test"
### 3.)  Consider the following; do they change the value printed for `A`?
+ A = ["problem"]
+ B = A[:]
+ B = "test"
